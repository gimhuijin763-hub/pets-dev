## 주요 이슈 및 해결

### 해결된 이슈

#### 1. JSX 파싱 오류 (2024-03-15)
**문제**: `app/landing/page.tsx`에서 "Unexpected token `div`" 오류 발생
**원인**: 숨은 특수문자 또는 깨진 JSX 구문
**해결**: 
- `app/landing/LandingPage.tsx` 새 파일 생성
- 깨끗한 JSX로 컴포넌트 재작성
- `app/landing/page.tsx`에서 re-export

#### 2. 이미지 표시 오류 (2024-03-15)
**문제**: 동물 상세 페이지에서 이미지가 표시되지 않음 (특히 "달이")
**원인**: localStorage 데이터에 `image_url` 필드 누락
**해결**:
- `utils/storage.ts`의 `getAnimals()` 함수 강화
- localStorage 데이터와 시드 데이터 병합
- 이미지 URL 누락 시 시드 데이터에서 복구
- `<img>` 태그에 `onError` fallback 추가

#### 3. 라우팅 혼선 (2024-03-15)
**문제**: `/`에서 동물 목록이 표시되고, 랜딩 페이지는 `/landing`에만 표시됨
**원인**: 라우팅 구조 설계 오류
**해결**:
- `app/page.tsx`를 랜딩 페이지 re-export로 변경
- `app/animals/page.tsx` 생성하여 동물 목록 이동
- Header 네비게이션 링크 업데이트 (`/` → `/animals`)
- 랜딩 페이지 CTA 링크 업데이트 (`/` → `/animals`)
